- Create a new cargo project
```sh
$ cargo new hello_cargo
```

- Separate with compile and execute
```sh
$ cargo build
$ ./target/debug/hello_cargo
```

- Compile + execute
```sh
$ cargo run
```

- Only check if it's okay to compile (not create binary)
```sh
$ cargo check
```

- Release build
```sh
$ cargo build --release
```
